# Sistema de Gestión en C++ con Qt

## 🇪🇸 Español

### Descripción
Este proyecto es un sistema de gestión desarrollado en **C++ con Qt Framework**, orientado al manejo de socios, permitiendo realizar operaciones como agregar, editar, eliminar, buscar y exportar registros.

El sistema también incluye un dashboard con estadísticas básicas y un sistema de logs para auditoría de cambios.

---

### Características principales

- Registro de socios (ID, nombre, saldo)
- Edición y eliminación de registros
- Búsqueda en tiempo real
- Exportación a archivo CSV (Excel)
- Dashboard con métricas:
  - Total de socios
  - Suma de saldos
  - Promedio de saldos
- Sistema de logs de acciones
- Interfaz gráfica con Qt
- Diseño moderno en modo oscuro

---

### Tecnologías utilizadas

- C++
- Qt Framework
- CMake
- STL (Vectors, Strings, Files)

🇺🇸 English

### Description

This project is a member management system developed in C++ with Qt Framework, focused on handling records using CRUD operations (Create, Read, Update, Delete).

It includes real-time search, CSV export, a statistics dashboard, and a logging system for tracking actions.

---

### Main Features

- Member registration (ID, name, balance)
- Edit and delete records
- Real-time search
- Export to CSV (Excel compatible)
- Dashboard with metrics:
  - Total members
  - Total balance
  - Average balance
- Action logging system
- Modern Qt GUI
- Dark professional theme

---

### Technologies Used

- C++
- Qt Framework
- CMake
- STL (vectors, strings, file handling)
